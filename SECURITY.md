# Security Policy

## Supported Versions

Security fixes are provided for the latest commit on `main` and for the most
recent released series.

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |
| < 0.1.0 | No |

## Reporting a Vulnerability

Please report security issues privately by email:

- `cjc44020@126.com`

When possible, include:

- affected version
- reproduction steps
- expected impact
- whether the issue affects the generated portable bundle, the builder itself,
  or both

Do not open a public GitHub issue for an undisclosed vulnerability.
